# Le Site du Nico

Bienvenue sur le repo qui contient tous les fichiers nécessaires pour mon super site bientôt exposé au Louvre !  

Le lien heroku : https://le-site-du-nicoc.herokuapp.com/

## Prérequis

Ruby 2.5.1
Rails 5.2.0
HTML 5
CSS 3

## En local

```git clone```
```bundle install```
```rails db:create```
```rails server```
